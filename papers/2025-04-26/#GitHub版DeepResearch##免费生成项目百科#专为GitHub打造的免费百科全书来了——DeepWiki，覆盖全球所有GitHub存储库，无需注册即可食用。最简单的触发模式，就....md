## AI 摘要

DeepWiki是由Cognition推出的免费GitHub项目百科工具，用户只需将GitHub链接中的"github"改为"deepwiki"即可自动生成项目专属百科页面。该工具提供可视化架构图、清晰的项目概述和交互式问答功能，目前已索引3万个存储库，处理40多亿行代码。其特色包括分层理解代码结构、一键生成技术文档，并支持AI对话查询。虽然因访问量过大曾导致服务器崩溃，但团队仍坚持免费开放使用。开发者可通过搜索或修改URL两种方式快速获取项目技术细节，显著提升代码库查阅效率。
