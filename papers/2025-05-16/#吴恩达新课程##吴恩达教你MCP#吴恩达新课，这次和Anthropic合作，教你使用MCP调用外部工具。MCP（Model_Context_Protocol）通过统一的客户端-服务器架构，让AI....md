## AI 摘要

吴恩达与Anthropic合作推出新课程《MCP构建丰富上下文的AI应用》，教授如何使用Model Context Protocol（MCP）统一接入外部工具和数据。课程时长不到2小时，涵盖MCP架构解析、搭建MCP服务器/客户端、连接参考服务及部署等实操内容，通过构建论文搜索机器人等案例演示如何实现AI动态调用工具。该协议采用客户端-服务器架构，简化AI集成外部资源流程。课程由Anthropic技术教育负责人主讲，现可在DeepLearning.AI平台免费学习，适合开发者提升AI工具调用能力。
