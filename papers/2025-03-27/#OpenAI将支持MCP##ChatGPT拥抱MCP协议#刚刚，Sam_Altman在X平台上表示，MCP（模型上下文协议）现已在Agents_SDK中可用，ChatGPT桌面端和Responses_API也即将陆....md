## AI 摘要

OpenAI宣布将在ChatGPT桌面端和Responses API中支持MCP（模型上下文协议）。MCP是Anthropic推出的开放标准，可将AI模型与外部数据源双向连接，实现实时数据交互。该协议已被Replit等公司采用，OpenAI的加入进一步巩固了其作为AI集成标准的地位。MCP能让AI助手直接访问业务工具和内容库，生成更准确的响应。Anthropic高管表示，MCP已成为拥有数千集成的开放标准。这一进展将使AI助手更好地理解用户实时数据，提供更贴心的服务。
