## AI 摘要

谷歌发布了开源的"A2A协议"（Agent版MCP协议），旨在促进不同Agent之间的自由协作。该协议支持跨平台和跨厂商部署，能够简化大规模多Agent系统的构建。这一举措将提升Agent间的互操作性，为开发者提供更便捷的多Agent系统开发环境。
