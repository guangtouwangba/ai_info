## AI 摘要

GitHub近期疑似对中国IP采取屏蔽措施：未登录用户访问时直接返回403错误，已登录用户仍可正常使用。经测试，中国大陆IP在未登录状态下多数页面无法访问。此前GitHub在中国处于不稳定访问状态，常需通过加速器或修改hosts访问。此次限制进一步升级，未登录用户完全无法访问主站，大幅提高使用门槛。目前GitHub官方尚未对此事作出回应。开发者社区猜测这可能是平台主动实施的IP屏蔽行为。
