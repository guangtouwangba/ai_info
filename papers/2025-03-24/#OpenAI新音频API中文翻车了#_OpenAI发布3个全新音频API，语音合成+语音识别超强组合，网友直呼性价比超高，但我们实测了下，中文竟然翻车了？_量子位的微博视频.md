## AI 摘要

OpenAI发布了三款全新音频API，包含语音合成和语音识别功能，性价比受到网友好评。但实测显示，这些API在处理中文时出现明显问题，表现不佳。具体问题未详细说明，但中文支持显然存在缺陷，导致"翻车"现象。这表明虽然新API在整体功能上有所突破，但对中文等特定语言的支持仍需改进。
