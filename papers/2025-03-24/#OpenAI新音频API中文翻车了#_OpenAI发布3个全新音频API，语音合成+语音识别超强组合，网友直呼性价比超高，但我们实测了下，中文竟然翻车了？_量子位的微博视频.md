## AI 摘要

OpenAI发布了三款全新音频API，提供语音合成和语音识别功能，性价比受到网友好评。然而实测发现，其中文语音合成效果不佳，出现明显"翻车"情况。相关测试视频显示，API在中文处理上存在明显问题，与英文表现形成反差。该事件引发了对OpenAI多语言处理能力的讨论。
